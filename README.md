# Evil8 Helm Charts

Public Helm charts published to `ghcr.io/evil8io/charts`.

## Usage

```bash
helm install <release> oci://ghcr.io/evil8io/charts/<chart>
```

## Charts

* [simple-cronjob](charts/simple-cronjob) — Run a script on a schedule
* [simple-hook](charts/simple-hook) — Run a Helm hook as a Job
